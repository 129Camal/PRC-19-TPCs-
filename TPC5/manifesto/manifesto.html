Abstract:

Neste TPC era pretendido fazer uma aplicação Web que fosse capaz de mostrar todos os repositórios no GraphDB.

O utilizador pode selecionar qual o repositório que pretende interrogar. Após essa escolha surge uma página com o número de triplos total, inferidos e explicitos. Para além disso, aparece um formulário para fazer a query em Sparql.

Após a query ser executada o resultado surge em baixo numa tabela.

Para efetuar os pedidos com a query, é efetuado o encode da query presente no formulário e adicionado ao pedido efetuado pelo Axios.

Deliverables:

  • Rotas da Aplicação Web;
  • Template da página de um repositório;
  • Template do indice de repositórios;
